User - користувач



Guest - незареєстрований користувач або користувач,

що не ввійшов у обліковий запис



Authorizated user - користувач, що увійшов у обліковий запис



(Надалі під "користувачем" ми матимемо на увазі "авторизоавного")



App - мобільний додаток для твіттера



Web page - сторінка в браузері для твіттера



User profile - головна стрінка користувача



Tweet - повідомлення на сторінці користувача



Retweet - можливість прокоментувати повідомлення іншого користувача 
і
відобразити у себе на сторінці



Comment - коментар



Like - спосіб вподобанння користувачами твіта або ін.



Lists - група рекомендованих облікових записів



Moments - це відібрана інформація у формі твітів про те,

що відбувається в світі



Follower - користувачі, що слідкують за сторінкою або користувачем



Following - сторінки або користувачі, за якими слідкує користувач



Dirrect message - особисті переписки користувача
